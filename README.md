# DeepSeek-Chat-For-WordPress
a deepseek chat plugin on wordpress website
插件使用介绍
功能：

添加聊天窗口，支持普通对话和深度思考模式。

查询剩余Token数量。

每日深度思考模式限制为5次，显示剩余次数。

限制每次对话长度为800字符。

使用方法：

将代码上传到/wp-content/plugins/文件夹下。

在WordPress后台启用插件。

编辑deepseek-chat.js文件，将API密钥添加到 API_KEY 变量中。

将网站发布到互联网上，即可使用聊天功能。
